## 2-js里的对象

### 对象的创建
三种方式
* 1. 字面量 
* 2. 构造函数 
* 3. Object.create()，属性方法放在原型上

------

**1、字面量方式**
```js
let obj = {
  name: '张三',
  age: 20
}
```

**2、构造函数方式**
```js
let obj = new Object()
obj.name = '张三'
obj.age = 20
```

**3、Object.create()方式，属性方法放在原型上**
```js
let obj = Object.create({
  name: '张三',
  age: 20
})
```

### 对象的调用
```js
// 点
obj.name

// 下标
obj['name']

// 变量
let str = 'name'
obj[str]
```

## 3-工厂模式
```js
// 类
function Person(name, age) {
  // 原料
  let obj = {}
  obj.name = name
  obj.age = age
  // 加工原料
  return obj // 出厂
}
let zhangsan = Person('张三', 20)
```

## 5-new运算符
new作用
* 1、运行函数
* 2、自动创建一个空对象
* 3、把空对象和this绑定
* 4、如果没有返回值，隐式返回this
-----

**1、运行函数**
```js
function fn() {
  console.log('fn')
}
fn()
new fn()
```

**2、自动创建一个空对象**<br>
**3、把空对象和this绑定**<br>
**4、如果没有返回值，隐式返回this**<br>
```js
function Test() {
  // let obj = {} // === this
  // ...
  // return this
}
new Test()
```

```js
// 工厂模式
function Person(name,age) {
  // let obj = {} //原料 === this
  this.name = name
  this.age = age
  // 加工原料
  // return obj // 出厂
}
let zhangsan = new Person('张三',20,'篮球')
console.log(zhangsan.name)
```
