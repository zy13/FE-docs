## 辅导员考试备考（面试篇）
```bash
    通过笔试，下一阶段就是面试环节，不同学校面试形式也有所不同，大体可以分为以下几类型：
1、一轮结构化面试；
2、两轮结构化面试；
3、两轮结构化面试+一轮实习考察；
4、无领导小组面试+结构化面试。

    首先要明确一个概念，通常面试一般会占到 50%，或者60% ，有些甚至会占70%，所以，尽管
笔试成绩不高或者排名靠后，但是完全可以通过面试将分数拉回来的，从而获得晋级名额，实际面试环节中，
这样的案例比比皆是，所以，不要因为笔试分数不高就选择放弃，要做的就是安心准备面试。

`01 着装准备`

    男生着装，个人建议：可以穿长袖的蓝（白）衬衫，黑西裤打领带，胸前佩戴一枚党徽，这应该是
相当不错的着装。
    女生着装，淡妆，不一定要正装出席，浅黄色等裙子搭配也是非常好的选择，而且对于评委正装审美
疲劳后，不同风格也会眼前一亮。

`02 结构化面试`

    首先，什么是结构化面试？
    面试程序和时间安排有结构，考官构成有结构，评分方式有结构，测评要素有结构，这种面试形式就是
结构化面试，常见于公务员、事业单位面试。

下面重点来说说面试题型，总结归纳高校辅导员面试题型，可以分为三类：
    `一、案例分析题`。
    例如：你的学生小刘最近在网站和论坛频频发帖，声讨家乡父母的房子受到了政府强拆一事，许多同学
义愤填膺，言语冲动，甚至要组队到他老家当地政府上访。你作为辅导员该怎么处理？

    `二、观点态度题`。
    例如：辅导员工作需要爱心，耐心和恒心，对此，你怎么看？

    `三、活动组织类题目`。
    例如：学校让二级学院开展“中国梦·我的梦”主题教育系列活动，你所在学院领导让你负责策划，
你会怎么做？

    如果有两轮结构化面试，通常会在这三类题型中选择两类进行考察。
    回答时间：3-5分钟，而是否有时间可以做思考需要自己提前了解，因为即兴作答和准备3分钟再作答
难度明显不一样，而这也需要提前做好准备。这三类题型中，观点态度题会稍难一些，但总体都是可以通过
不断加强训练做到形态优雅，言之有理，言之有物。

    有关如何训练的问题，之前有个成功考上公务员的朋友和我分享过这样一个经验： `当时他们5个通过
公务员笔试的朋友一起抱团，带着自己所收集到的面试题目，互相模拟，互相指正，最后做到的程度就是当
题目一出来，结构化的回答就是一种本能思维。`当然，这其中肯定是要多总结相同类型的题目，归纳答法，
总结理论，同时不断地进行加强训练，经过一段时间，无论在表达上还是思维上，肯定会有个质的飞跃。
    例如，活动组织类真题：领导让你组织开展一场心理危机干预讲座，你该怎么做？
    参考：开展讲座是高校辅导员的一项常规工作，当我负责组织这场心理危机干预的讲座时，我会按照
以下流程开展。
    `第一，确定参加的人员及规模`。跟领导提前确定需要邀请人员以及举办讲座的学生受众，以便后期
邀请函的制作以及通知的发放。对于要邀请的人员，需要至少提前一周联系确定是否能够出息，现场准备好
嘉宾牌。对于参加的学生受众，通知学院心理组织学生干部、心理委员、宿舍长等积极参加，而其他同学
根据自己的兴趣选择性参加相结合的方式，并做好相关登记。
    `第二，邀请讲座主讲人`。一般会优先邀请跟所在学院挂钩的校心理健康中心的指导老师，他们对于
学院学生情况比较了解，如果他们时间不能错开，其次再考虑邀请校内外在心理危机干预做的较好的心理
健康工作者。同时，跟邀请的主讲人协商好讲座开展的时间以及具体的题目，时间一般会安排在晚上或者
周末，并充分考虑是否和学校的大型活动以及学生课程等冲突。
   ` 第三，做好后勤工作`。组织人员，合理分工，分别进行场地的借用，海报宣传单以及邀请函的制作
以及发放，现场设备的调试以及讲座中间游戏设计等工作。
    `第四，讲座流程的安排`。确定是否开场需要领导讲话，提前准备讲话稿以及讲座结束后问题的解答
等等提前跟与会相关人员做好沟通，确保讲座有序顺畅。
    `第五，跟进全程并做好宣传报道`。全程做好讲座跟进，出现问题及时进行反馈，讲座过程中做好
摄影并在结束当天写好新闻稿，审核完毕后通过院系网站、微信公众号等全媒报道。
    我会按照以上的工作思路，结合实际预算，提交策划书给领导审核，按照流程认真细致的完成工作，
谢谢！


`03 实习考察面试`

    大家对于这一环节可能会有疑问，实习考察具体会是些什么内容呢？ 其实就是分配到学院进行
具体实操实习，作为实习辅导员，一般只有一个星期的实习时间，是没有具体负责学生的，所以这时会
负责的主要是些文书类工作，比如写活动策划书，撰写新闻稿，思考心理工作具体实施方案以及协助学院
举办相关活动。
    这时可能有部分考生就会有所顾虑了，我之前没有做过这些类型的工作，感觉会做不好。这里要分享的是
通常去实习，氛围会很和谐，所以放开手脚去做就行啦，与此同时，做的质量是考察的一部分，但更为重要的是
考生的个人态度，懂不懂得反馈，遇到难题会不会沟通解决以及积极性如何等等这些因素。

```

```bash
前几天有好几位同学问我，公共基础知识应该如何复习。我踌躇了半天，最后
还是只能说出两个关键词——积累、刷题。公共基础知识较行测而言，板块更
多，知识点更松散，包括政治、历史、法律、管理、科技、人文历史、国情、
地理等，前者复习主要是在熟悉题目类型、了解解题思路，后者更多的是积
累。网上有很多题目，一些公考的公众号上面也有每日一练，大家可以自己找
来做做。
```
